# HealthWell Template - Design Reference

## Overview
HealthWell is a modern, clean healthcare template with a minimalist aesthetic.

## Color Palette
- **Primary Green:** #46b96e (buttons, CTAs, accent sections)
- **Secondary Green (darker):** #389458
- **Cyan Accent:** #0bf
- **Orange Accent:** #f29940
- **Background White:** #ffffff
- **Background Off-white:** #f7f7f7, #f4fbf6
- **Text Dark:** #000000, #333333
- **Text Gray:** #666666, #999999
- **Border Gray:** #cccccc, #e5e5e5

## Typography
- **Font Family:** Inter (weights: 200, 300, 400, 500, 600, 700, 900)
- **Headlines:** Bold, large (48-72px on desktop)
- **Subheadings:** Semi-bold, medium (24-32px)
- **Body Text:** Regular, 16-18px
- **Small Text:** 14px

## Layout Patterns
- **Max Width:** 1920px (desktop), 1200px content area
- **Padding:** 12px-96px responsive
- **Section Spacing:** Large whitespace between sections
- **Grid:** Flexbox-based with CSS custom properties

## Responsive Breakpoints
- **Desktop:** 1200px+
- **Tablet:** 810-1199px
- **Mobile:** <810px

## Component Patterns

### Buttons
- Primary: Green background, white text, rounded corners
- Secondary: White/transparent background, green border, green text
- Hover: Darker shade

### Cards
- White background
- Subtle shadow or border
- Rounded corners (8-12px)
- Padding: 24-32px

### Form Inputs
- Clean border
- Rounded corners
- Focus state with green accent

### Icons
- Simple, line-style
- Consistent stroke width
- Green accent color when needed

### Images
- High-quality photography
- Rounded corners on some
- Natural, healthcare-appropriate subjects
